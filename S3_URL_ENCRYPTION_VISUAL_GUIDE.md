### Scenario 2: Network Sniffer (HTTP) âŒ â†’ âœ…

**Before (without HTTPS):**
```
Attacker sniffs HTTP traffic:

- Bucket name
- Region
- S3 path structure
- Can potentially replicate requests

**After (with HTTPS + Encryption):**
```
Attacker sniffs HTTPS traffic:
GET /api/images/proxy/a1b2c3d4e5f6:f6e5d4c3b2a1...

Attacker sees:
- Encrypted token (useless without key)
- HTTPS prevents plaintext sniffing
- Token expires after 24 hours
- Different token each time
```

```
